# Certificates page

The plugin also exposes a dedicated certificates page.

## What it shows

- proofs that already expose a certificate link
- the certificate URL for each proof
- a shortcut to the proof detail page

## Navigation

By default the page is registered as:

- label: `Certificates`
- slug: `/admin/proovit/certificates`
- navigation group: `ProovIT`

## Notes

- the page is filtered to proofs that already have a certificate URL
- it is intended as a quick access screen for downloaded or opened certificates
