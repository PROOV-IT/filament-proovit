# Proof detail page

The plugin includes a dedicated proof view page for inspecting a single proof.

## What it shows

- proof name and title
- sequence number
- human-readable status
- signed timestamp
- certificate URL when available
- description, metadata, files, and history in a read-only layout

## Actions

- refresh the proof data
- open the certificate link when it exists
- revoke the proof when the workflow allows it
- inspect the attached files and their download links

## Notes

- the page is not part of the main navigation
- it is usually opened from the proofs or certificates table actions
